## math.js Functions

Below are the functions included in `math.js` and what they do:

- `double(n)`: Returns n multiplied by 2.
- `square(n)`: Returns n multiplied by itself.
- `isEven(n)`: Returns true if the number is even, otherwise false.
- `isOdd(n)`: Returns true if the number is odd, otherwise false.
- `multiply(a, b)`: Returns the product of a and b.
- `half(n)`: Returns n divided by 2.
